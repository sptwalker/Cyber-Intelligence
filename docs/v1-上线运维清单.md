## 二、部署清单（把"我的电脑"变成"常驻服务器"）

1. **一台国内云服务器**（已备案；能跑 Chrome + Python），装 opencli + 本项目。
2. **专用可牺牲监控账号**（微博/知乎）——**绝不用真人主账号**（封号连坐），profile 与真人隔离。
3. **常驻调度**：`python -m yuqing.scheduler [间隔分钟]`（用 systemd/nssm 托管，或 cron 直接调 `scheduler once`）。
4. **deadman 外部探活**：另设一个 cron，每小时 `python -m yuqing.scheduler deadman`——调度进程若挂了，这个外部 cron 会推飞书叫人（进程自己挂了报不了自己）。
5. **环境变量**：`ANTHROPIC_API_KEY`（Claude 深抽，否则规则兜底）、`FEISHU_WEBHOOK`（推送）、`YUQING_OPENCLI_SESSION`、`YUQING_MAX_CALLS/TOKENS`（成本熔断上限）。
6. **备份**：定期备份 `yuqing.db`（WAL 模式，连 `-wal/-shm` 一起或先 checkpoint）。
